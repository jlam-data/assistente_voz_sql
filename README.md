# assistente_voz_sql
Trata-se de um Assistente de voz que roda comandos SQL e explica as queries em linguagem amigável ao usuário. Apresenta ainda possibilidades de desenvolvimento futuro para interoperabilidade com bancos de dados, quando estiver vinculado a um negócio real. Assim, poderia trazer de volta os dados, além da informação sobre comandos.
